Brief tour on how did we arrive to here.

  1. Pre-compute the statistics and contour plots of all results
  2. Group them into a big json file
  3. Use a plotting script to read and plot them -> the big contour plots
  4. From the same json file, extract the statistics part and format them as LaTeX tables, and save them into txt files.
  5. Use MathJax to compile those LaTeX tables in HTML.
  6. Plot those kicks with Plotly, export them as HTML div blocks.
  7. Import them into this same HTML with js.
